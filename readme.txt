1. Download tensorflow, keras, pandas,numpy,nltk,scikit on your computer in-order
   to run the files
2. Do not touch the toxic_filter.py and the model.h5 files
3. Download the test and train dataset from https://www.kaggle.com/c/jigsaw-toxic-comment-classification-challenge/data
   and keep it in the same directory as the prreviously downloaded files.
4. In-order to access the filter, open the nlp_test.py file
5. The code allows the user to enter a comment and displays
   whether a comment is toxic or not along with it's toxicity probability.
